# Installation

Get started with Vera UI, a modern React component library built with accessibility and customization in mind.

## Requirements

<Callout type="info">
  Make sure you have Node.js 18+ and a React project set up before installing Vera UI.
</Callout>

- **React**: ^18.2.0 || ^19.0.0
- **React DOM**: ^18.2.0 || ^19.0.0  
- **Node.js**: >=18

## Install the Package

<Steps>

### Install Vera UI

```bash npm2yarn
npm install @helgadigitals/vera-ui
```

### Install Peer Dependencies

Depending on your needs, you may need to install additional peer dependencies:

```bash npm2yarn
# For forms
npm install react-hook-form zod

# For routing (optional)
npm install react-router-dom

# For advanced table features
npm install @tanstack/react-table

# For date handling
npm install date-fns
```

### Set Up Tailwind CSS

Vera UI is built with Tailwind CSS. You need to set up Tailwind in your project:

```bash npm2yarn
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Configure Tailwind CSS

### Import Vera UI Styles

Finally, import Vera UI's CSS in your main file:

```tsx title="src/main.tsx" {3}
import React from 'react'
import ReactDOM from 'react-dom/client'
import '@helgadigitals/vera-ui/styles'
import './globals.css'
import App from './App'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
)
```

</Steps>

## Quick Start

Now you can start using Vera UI components in your React application:

```tsx title="src/App.tsx"
import { Button, Card, CardContent, CardHeader, CardTitle } from '@helgadigitals/vera-ui'

function App() {
  return (
    <div className="p-8">
      <Card className="w-96">
        <CardHeader>
          <CardTitle>Welcome to Vera UI</CardTitle>
        </CardHeader>
        <CardContent>
          <p className="mb-4 text-muted-foreground">
            A modern React component library built for accessibility and customization.
          </p>
          <Button>Get Started</Button>
        </CardContent>
      </Card>
    </div>
  )
}

export default App
```
