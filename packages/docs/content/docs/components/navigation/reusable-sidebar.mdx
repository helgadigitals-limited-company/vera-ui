# Reusable Sidebar

The `ReusableSidebar` component provides a comprehensive navigation solution with support for simple items, grouped navigation, collapsible modes, and responsive behavior. It's built on Radix UI primitives and follows the Vera UI design system.

### SidebarProvider

The provider component manages sidebar state and must wrap the `ReusableSidebar` :

```tsx
import { SidebarProvider } from '@helgadigitals/vera-ui'
<SidebarProvider>
  <ReusableSidebar items={items} />
</SidebarProvider>
```

<Callout type="warning">
**Important**: The `ReusableSidebar` component must be wrapped with `SidebarProvider` to function properly.
</Callout>


## Installation

```bash npm2yarn
npm install @helgadigitals/vera-ui lucide-react
```

## Sidebar Features

### Responsive Behavior
- **Desktop**: Full sidebar with collapsible icon mode
- **Tablet & Mobile**: Overlay behavior with mobile-friendly touch controls
- **Automatic State Management**: Built-in responsive breakpoint detection

### Navigation Types
- **Simple Items**: Direct navigation links with icons and optional badges
- **Grouped Items**: Organized sections with collapsible group headers
- **Mixed Content**: Combination of simple items and grouped sections

### Built-in Functionality
- **Active State Detection**: Automatically highlights current route
- **Keyboard Navigation**: Full keyboard accessibility support
- **Dark Mode Support**: Respects theme system preferences
- **Icon Integration**: Built for Lucide React icons

## Router Integration

The sidebar automatically detects active routes based on the current pathname. It works with any router that updates `window.location.pathname`.

## Props Reference

### ReusableSidebar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `Array<SidebarItem \| Group>` | - | Navigation items configuration |
| `heading` | `string` | - | Sidebar header title |
| `image` | `string` | - | Logo/image URL for header |
| `isFooterVisible` | `boolean` | `false` | Show footer with user info |
| `label` | `string` | - | Group label for navigation items |
| `displayName` | `string` | - | User display name in footer |
| `classNames` | `ReusableSidebarClassNames` | - | Custom CSS classes for different parts |
| `stylesConfig` | `SidebarStylesConfig` | - | Style configuration object |
| `style` | `React.CSSProperties` | - | Inline styles |

### SidebarProvider

The provider component manages sidebar state and must wrap the `ReusableSidebar`:

```tsx
<SidebarProvider>
  <ReusableSidebar items={items} />
</SidebarProvider>
```

## Type Definitions

### SidebarItem
```tsx
interface SidebarItem {
  title: string          // Display text for the navigation item
  path: string           // Navigation path/route
  icon: LucideIcon       // Icon component from lucide-react
  badge?: string | number // Optional badge content (notifications, counts, etc.)
}
```

### Styling Configuration
```tsx
interface SidebarStylesConfig {
  bgColor?: string          // Background color Tailwind class
  textColor?: string        // Text color Tailwind class
  widthClass?: string       // Width Tailwind class
  itemTextSize?: string     // Item text size class
  headingTextSize?: string  // Heading text size class
  surfaceClasses?: string   // Surface styling classes
}

interface ReusableSidebarClassNames {
  root?: string               
  header?: string              
  image?: string               
  heading?: string            
  groupLabel?: string         
  menu?: string               
  menuItem?: string            
  menuButton?: string         
  menuButtonActive?: string    
  footer?: string             
  footerUserIcon?: string      
  footerDisplayName?: string   
}
