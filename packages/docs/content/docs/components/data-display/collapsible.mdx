# Collapsible

A small wrapper around `@radix-ui/react-collapsible` that provides three exported components:

- `Collapsible` — the Root (container)
- `CollapsibleTrigger` — the button/element that toggles open/closed
- `CollapsibleContent` — the content shown when open

<CollapsibleExample />

## Quick usage

```tsx
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from './collapsible'

<Collapsible defaultOpen={false}>
  <CollapsibleTrigger asChild>
    <button>Show details</button>
  </CollapsibleTrigger>

  <CollapsibleContent>
    <div>
      {/* your content */}
    </div>
  </CollapsibleContent>
</Collapsible>
