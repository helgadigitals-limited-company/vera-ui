## What is Vera UI?

Vera UI is a comprehensive React component library built with modern web standards and accessibility in mind. It provides a complete set of UI primitives and complex components to help you build professional applications quickly and efficiently.

<Callout type="info">
**Built for Scale**: Vera UI is designed to grow with your project, from simple prototypes to complex enterprise applications.
</Callout>

## Quick Start

<Steps>
  <Step>
    ### Install Vera UI
    
    Add Vera UI to your React project:
    
    ```bash npm2yarn
    npm install @helgadigitals/vera-ui
    ```
  </Step>
  
  <Step>
    ### Setup Theme Provider
    
    Wrap your app with the theme provider:
    
    ```tsx
    import { ThemeProvider } from "@helgadigitals/vera-ui";
    
    function App() {
      return (
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
        >
          <YourApp />
        </ThemeProvider>
      );
    }
    ```
  </Step>
  
  <Step>
    ### Start Building
    
    Import and use components:
    
    ```tsx
    import { Button, Card, CardHeader, CardTitle, CardContent } from "@helgadigitals/vera-ui";
    
    function Welcome() {
      return (
        <Card className="w-96">
          <CardHeader>
            <CardTitle>Welcome to Vera UI</CardTitle>
          </CardHeader>
          <CardContent>
            <p>Start building beautiful interfaces!</p>
            <Button className="mt-4">Get Started</Button>
          </CardContent>
        </Card>
      );
    }
    ```
  </Step>
</Steps>

## Why Choose Vera UI?

### Developer Experience
- **IntelliSense Support**: Full TypeScript definitions for better development
- **Consistent API**: Predictable component interfaces across the library
- **Easy Integration**: Works seamlessly with existing React applications
- **Comprehensive Documentation**: Detailed guides and examples for every component

### Production Ready
- **Battle Tested**: Used in production applications with thousands of users
- **Performance Optimized**: Minimal bundle size with tree-shaking support
- **SSR Compatible**: Works with Next.js, Gatsby, and other SSR frameworks
- **Cross-Browser Support**: Tested across modern browsers and devices

### Future Proof
- **Regular Updates**: Continuous improvements and new features
- **Community Driven**: Open to feedback and contributions
- **Standards Compliant**: Built with web standards and best practices
- **Framework Agnostic**: Core principles can be adapted to other frameworks

## Advanced Features

### Flexible Sidebar System
One of Vera UI's standout features is the advanced sidebar navigation system:

```tsx
import { SidebarLayout, type MixedSidebarItem } from "@helgadigitals/vera-ui";
import { Home, Users, Settings, FileText } from "lucide-react";

const navigationItems: MixedSidebarItem[] = [
  { title: "Dashboard", path: "/", icon: Home },
  { title: "Quick Actions", path: "/quick", icon: Zap, badge: "New" },
  {
    key: "content",
    label: "Content Management",
    items: [
      { title: "Posts", path: "/posts", icon: FileText },
      { title: "Pages", path: "/pages", icon: FileText }
    ]
  },
  {
    key: "admin",
    label: "Administration",
    items: [
      { title: "Users", path: "/users", icon: Users, badge: 12 },
      { title: "Settings", path: "/settings", icon: Settings }
    ]
  }
];

function App() {
  return (
    <SidebarLayout props={{ items: navigationItems }}>
      <YourAppContent />
    </SidebarLayout>
  );
}
```

### Advanced DataTable
Build complex data interfaces with the powerful DataTable component:

```tsx
import { DataTable } from "@helgadigitals/vera-ui";

<DataTable

tableData={users}
  tableColumns={['name', 'email', 'role', 'status']}
  sortableColumns={['name', 'email', 'role']}
  enableGlobalFilter
  selectable
  actions={[
    {
      id: "edit",
      label: "Edit User",
      icon: <Edit className="h-4 w-4" />,
      onClick: (row) => handleEdit(row)
    }
  ]}
/>
```
