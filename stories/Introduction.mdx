# Welcome to Vera UI

Vera UI is a comprehensive React component library built with **Radix UI** primitives and styled with **Tailwind CSS**. It provides a collection of accessible, customizable, and production-ready components that follow modern design principles.

## ✨ Features

- **🎨 Beautiful Design**: Modern, clean design system with consistent visual language
- **♿ Accessibility First**: Built with Radix UI primitives ensuring WCAG compliance
- **🌗 Dark Mode Ready**: Full support for light and dark themes
- **📱 Responsive**: Mobile-first design with responsive components
- **🎯 TypeScript**: Full TypeScript support with comprehensive type definitions
- **⚡ Performance**: Optimized for performance with minimal bundle size
- **🔧 Customizable**: Easy to customize with CSS variables and Tailwind classes

## 🏗️ Built With

- **React 19** - The latest version of React
- **Radix UI** - Unstyled, accessible components
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool and development server
- **Class Variance Authority** - For component variants
- **Lucide React** - Beautiful & consistent icons

## 🎯 Philosophy

Vera UI is designed with the following principles in mind:

1. **Accessibility is not optional** - Every component is built with accessibility in mind
2. **Developer Experience** - Simple APIs with excellent TypeScript support
3. **Customization** - Easy to theme and customize to match your brand
4. **Performance** - Fast, lightweight components with minimal overhead
5. **Consistency** - Cohesive design language across all components
