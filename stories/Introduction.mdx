# Welcome to Vera UI

`@helgadigitals/vera-ui` is a modern React component library built on **Radix UI** + **Tailwind CSS** with strong TypeScript support and theming primitives.

## ✨ Features

- 🎨 Consistent design tokens & theming
- ♿ Accessibility via Radix primitives
- 🌗 Dark mode ready
- 📱 Responsive + mobile aware (drawers, sheets)
- ⚡ Tree‑shakable ES modules
- 🧪 TypeScript-first, strict types
- 🔌 Composable + headless patterns

## 🏗️ Core Stack

| Layer | Tech |
| ----- | ---- |
| Framework | React 18 / 19 |
| Primitives | Radix UI |
| Styling | Tailwind CSS |
| Forms | react-hook-form (+ zod resolvers) |
| Data | @tanstack/react-table |
| Validation | zod |

## 📦 Packaging Model

The library marks critical singletons as `peerDependencies` (not bundled):

Required peers:
```
react
react-dom
@tanstack/react-table
react-hook-form
zod
date-fns
```

Optional peer:
```
react-router-dom
```

Install them (see Installation doc) to avoid duplicate React or context instances.

## 🧱 High-Level Architecture

- Low-level UI (buttons, inputs, overlays)
- Layout primitives (sidebar, panels)
- Form integrations (validators, RHF adapters)
- Theme provider + design tokens
- Composable hooks (state + accessibility helpers)

## 🎯 Philosophy

1. Accessibility first
2. Predictable APIs
3. Minimal runtime styling logic (Tailwind utilities)
4. Escape hatches for customization (classNames / theme prop)
5. No hidden singletons (explicit providers)

## 🔧 Customization Snapshot

```tsx
import { SidebarProvider } from "@helgadigitals/vera-ui"

<SidebarProvider
  theme={{
    bgColor: "bg-zinc-900",

    menuButtonActive: "bg-zinc-800/60",
  }}

  {/* app */}
</SidebarProvider>
```

See Installation for Tailwind config & CSS import details.

## 🚑 Troubleshooting (Quick)

| Symptom | Fix |
| ------ | --- |
| Invalid hook call | Ensure single React: `pnpm ls react` |
| Styles missing | Tailwind content paths include the package + imported CSS |
| Sidebar mobile crash | Missing peer deps / duplicate React build |
| Types missing | Confirm `dist/index.d.ts` resolved; restart TS server |

Full details in Installation doc.
